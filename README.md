# twstock — 台股即時報價監控工具

即時監控多支台灣股票報價，並在達到自訂到價時發出警示。

---

## 功能

- 每 5 秒自動刷新報價（可調整）
- 同時監控多支股票
- 到價警示：終端機紅字顯示 + 嗶聲提醒
- 接近到價（10% 內）：黃字提示
- 正常狀態：綠字顯示

---

## 安裝

```bash
pip install twstock lxml
```

---

## 使用方式

### 1. 編輯設定檔 `config.json`

首次執行若無 `config.json`，會自動建立範例檔案。

```json
{
  "interval": 5,
  "stocks": [
    {
      "id": "2330",
      "name": "台積電",
      "upper": 1000,
      "lower": 900
    },
    {
      "id": "2317",
      "name": "鴻海",
      "upper": 200,
      "lower": 150
    }
  ]
}
```

| 欄位 | 說明 | 必填 |
|------|------|------|
| `id` | 股票代號（台股四位數代號） | ✓ |
| `name` | 自訂顯示名稱 | 選填 |
| `upper` | 到價上限，超過此價格發出警示 | 選填 |
| `lower` | 到價下限，低於此價格發出警示 | 選填 |
| `interval` | 報價刷新間隔（秒），預設 5 | 選填 |

> `upper` / `lower` 可省略，省略則不監控該方向。

### 2. 啟動監控

```bash
python twstock_monitor.py
```

### 3. 結束程式

按 `Ctrl + C` 即可停止監控。

## 狀態說明

| 狀態 | 顏色 | 說明 |
|------|------|------|
| ✓ | 綠色 | 價格在正常範圍內 |
| ▲ 接近上限 | 黃色 | 現價 >= 上限 × 90% |
| ▼ 接近下限 | 黃色 | 現價 <= 下限 × 110% |
| ⚠ 已達上限! | 紅色 + 嗶聲 | 現價 >= 上限 |
| ⚠ 已達下限! | 紅色 + 嗶聲 | 現價 <= 下限 |
