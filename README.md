# twstock — 台股即時報價監控工具

即時監控多支台灣股票與期貨報價，並在達到自訂到價時發出警示。

---

## 工具一覽

| 腳本 | 用途 | 資料來源 |
|------|------|---------|
| `twstock_monitor.py` | 上市／上櫃股票即時報價 | twstock 套件（即時） |
| `futures_monitor.py` | 台指期 + 個股期貨報價 | TAIFEX OpenAPI（延遲） |

---

## 安裝

```bash
pip install twstock lxml
```

> `futures_monitor.py` 僅使用 Python 標準函式庫，不需額外安裝套件。

## 股票監控 — `twstock_monitor.py`

### 設定檔 `config.json`

首次執行若無 `config.json`，會自動建立範例檔案。

```json
{
  "interval": 5,
  "stocks": [
    {
      "id": "2330",
      "name": "台積電",
      "upper": 1900,
      "lower": 1760
    },
    {
      "id": "2317",
      "name": "鴻海",
      "upper": 200,
      "lower": 150
    }
  ]
}
```

| 欄位 | 說明 | 必填 |
|------|------|------|
| `id` | 股票代號（台股四位數代號） | ✓ |
| `name` | 自訂顯示名稱 | 選填 |
| `upper` | 到價上限，達到此價格發出警示 | 選填 |
| `lower` | 到價下限，跌到此價格發出警示 | 選填 |
| `interval` | 報價刷新間隔（秒），預設 5 | 選填 |

> `upper` / `lower` 可省略，省略則不監控該方向。

### 啟動

```bash
python twstock_monitor.py
```

### 畫面範例

```

----------------------------------------------------------------------

========================================================================
```

## 期貨監控 — `futures_monitor.py`

### 支援合約

- **台指期**：直接使用代號 `TX`
- **個股期貨**：填入股票代號（如 `2330`），程式自動對應期貨合約代碼

### 設定檔 `futures_config.json`

首次執行若無 `futures_config.json`，會自動建立範例檔案。

```json
{
  "interval": 30,
  "contracts": [
    {
      "id": "TX",
      "name": "台指期",
      "upper": 34000,
      "lower": 32000
    },
    {
      "id": "2330",
      "name": "台積電期",
      "upper": 1900,
      "lower": 1760
    }
  ]
}
```

| 欄位 | 說明 | 必填 |
|------|------|------|
| `id` | 台指期填 `TX`；個股期貨填股票代號（如 `2330`） | ✓ |
| `name` | 自訂顯示名稱 | 選填 |
| `upper` | 到價上限 | 選填 |
| `lower` | 到價下限 | 選填 |
| `interval` | 刷新間隔（秒），建議 30 以上 | 選填 |

### 啟動

```bash
python futures_monitor.py
```

### 畫面範例

```
================================================================================
  台灣期貨即時報價監控  |  更新時間: 2026-03-30 09:35:10  |  按 Ctrl+C 結束
================================================================================
代號   名稱       現價    開盤    最高    最低   漲跌   成交量  狀態
--------------------------------------------------------------------------------
TX     台指期    33120   32793   33243   32652   -321    64049  ✓
QFF    台積電期   1830    1810    1835    1805     -20    13467  ✓
================================================================================
  ※ 資料為延遲報價（非即時）
```

> 顯示代號為期貨合約代碼（如 QFF），非原始股票代號。

## 狀態說明（兩支工具共用）

| 狀態 | 顏色 | 說明 |
|------|------|------|
| ✓ | 綠色 | 價格在正常範圍內 |
| ▲ 接近上限 | 黃色 | 現價 ≥ 上限 − margin |
| ▼ 接近下限 | 黃色 | 現價 ≤ 下限 + margin |
| ⚠ 已達上限! | 紅色 + 嗶聲 | 現價 ≥ 上限 |
| ⚠ 已達下限! | 紅色 + 嗶聲 | 現價 ≤ 下限 |

**接近閾值 margin 規則：**

| 股價／期貨價格 | margin |
|--------------|--------|
| ≥ 1000 | 5（即上限 -5 / 下限 +5 觸發黃燈） |
| < 1000 | 1（即上限 -1 / 下限 +1 觸發黃燈） |

---

## 結束程式

按 `Ctrl + C` 即可停止監控。
